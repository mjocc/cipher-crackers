# cipher-crackers

## To install

### For production

Download latest release from releases on the right  
Install it with pip
```shell
$ pip install ciphertool-0.1.0.tar.gz
```

### For development

Create a virtual env and activate it
```shell
$ python -m venv env
$ source env/bin/activate
```

Install wheel
```shell
$ pip install wheel
```

Install package
```shell
$ pip install .
```

## To run

Use the command `ciphertool`  
For how to use
```shell
$  ciphertool --help
```
